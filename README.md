# Sculptr — Landing Site

Marketing site for Sculptr, coaching software for personal trainers.

## Run

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS 3
- Instrument Serif + Inter via `next/font`

## Philosophy

No icons. No generic SaaS vocabulary. The messaging is: the app adapts to you, not the other way round.
